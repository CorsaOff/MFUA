# Git Sync Automation Tool

Проект содержит скрипты для автоматической синхронизации между двумя локальными репозиториями.

## Описание работы
1. **Проверка путей:** Скрипт проверяет наличие указанных директорий перед началом работы.
2. **Обновление:** Выполняется `git pull` в исходном репозитории для получения актуальных данных.
3. **Синхронизация:** Файлы копируются из источника в целевую папку. Папка `.git` игнорируется, чтобы не повредить историю целевого репозитория.
4. **Push:** Изменения автоматически индексируются и отправляются в удаленную копию второго репозитория.

## Файлы
* `sync.sh` — скрипт для систем на базе Linux (Bash).
* `sync.ps1` — скрипт для Windows (PowerShell).
